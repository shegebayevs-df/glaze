# GLAZE — Модуль автономного сбора мусора с поверхности водоёмов

Проект разработан для онлайн-этапа республиканского хакатона Tech Vision 2026 (направление: Smart City & GreenTech, кейс: Вывоз и переработка мусора).

---

## 📌 Проблема и целевой пользователь

* **Целевой пользователь:** Службы благоустройства и операторы городских парковых водоёмов.
* **Одна острая боль:** Высокие трудозатраты и низкая скорость ручной очистки закрытых водоёмов от плавающего мелкого пластика и мусора.
* **Решение:** Прототип автономного катамарана GLAZE, объединяющий компьютерное зрение для обнаружения мусора на воде и микроконтроллерное управление ходовой частью и конвейером.

---

## 🛠️ Архитектура и технологический стек

Система состоит из двух модулей, связываемых по USB-Serial:

1. **Модуль детекции (main.py)**:
   * **Стек:** Python 3, OpenCV, Ultralytics YOLOv8, PySerial.
   * **Логика:** Считывает видеопоток с камеры, находит плавающий мусор, определяет его координаты относительно центра кадра и формирует пакет управляющих команд для движения и захвата.

2. **Аппаратный модуль (firmware/glaze_firmware.ino)**:
   * **Стек:** C++, Arduino IDE.
   * **Логика:** Прошивка микроконтроллера Arduino Uno R3. Принимает строки вида M<left>,<right>,<conveyor>\n, управляет ходовыми моторами и лентой конвейера через драйвер L298N, а также считывает показания датчиков.

## ⚙️ Инструкция по локальному запуску

### 1. Клонирование репозитория
git clone https://github.com/shegebayevs-df/glaze.git
cd glaze

### 2. Установка зависимостей Python
pip install -r requirements.txt

### 3. Прошивка микроконтроллера
1. Откройте файл firmware/glaze_firmware.ino в Arduino IDE.
2. Выберите плату Arduino Uno и соответствующий COM-порт.
3. Загрузите прошивку на плату.

### 4. Запуск управляющего скрипта
Подключите камеру и плату Arduino к компьютеру по USB, после чего запустите:
python main.py
